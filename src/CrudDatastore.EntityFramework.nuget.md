# CrudDatastore.EntityFramework

CrudDatastore.EntityFramework provides boilerplate templates and abstractions for seamless Entity Framework integration with the CrudDatastore framework.

## Features

- Pre-built abstractions for CRUD operations
- Specification pattern integration for complex queries
- Repository pattern abstractions
